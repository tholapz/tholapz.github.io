# tholapz.github.io

Personal journal and blog by Kamol, hosted at [www.tholapz.com](https://www.tholapz.com).

Built with [Jekyll](https://jekyllrb.com/) using the [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) theme and deployed via GitHub Pages.

## Topics

Writing covers finance, technology, AI, business, and occasional personal reflections on life, scuba diving, and everything in between.

## Local Development

```shell
bundle exec jekyll serve
```
